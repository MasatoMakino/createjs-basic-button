Class BasicRadioButtonManager

排他的に選択されるボタンを制御するクラスです。

メンバーのBasicRadioButtonが変更されると、このクラスに変更内容が通知されます。 このクラスは変更通知に応じ、他のBasicRadioButtonオブジェクトの選択状態を変更します。 また、このクラスは変更内容をEventとして発信します。

利用する際には以下のような手順でインスタンス化してください。

let btn:BasicRadioButton = new BasicRadioButton(...); let manager:BasicRadioButtonManager = new BasicRadioButtonManager(); //インスタンス化 manager.addButton(btn); manager.selected = btn; //デフォルトで選択されているボタンを指定
